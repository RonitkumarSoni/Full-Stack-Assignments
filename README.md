# Full Stack Coding Video Assignment

A professionally organized collection of 30 practical full stack coding assignments covering React, Node.js, Express.js, and MongoDB interview-focused problems.

This repository was created as part of a coding video assignment series where every solution is implemented, explained, and structured separately for better learning, recording, revision, and interview preparation.

The project focuses on writing clean and beginner-friendly code while covering important real-world development concepts commonly asked in technical interviews and coding rounds.

## How To Run The Projects

Open any assignment folder and install dependencies:

```bash
npm install
```

Run Node.js projects:

```bash
npm start
```

Run React + Vite projects:

```bash
npm run dev
```

MongoDB assignments require a local MongoDB server or a valid MongoDB Atlas connection string.
