# Full Stack Coding Video Assignment

This repository contains the completed code work for an advanced full stack coding video assignment focused on practical interview questions from React, Node.js, and MongoDB.

The assignment was completed as 30 short coding walkthroughs, with each solution built as a separate question folder for clean review, recording, and viva preparation.

## How To Run

Open any question folder and install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

For React/Vite projects, use:

```bash
npm run dev
```

MongoDB-based questions require a local MongoDB server or a valid MongoDB connection string.
